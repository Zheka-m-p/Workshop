# Morse Code Trainer

Консольный тренажёр для тренировки восприятия азбуки Морзе. Программа показывает случайные слова, закодированные в последовательности точек и тире, и предлагает угадать их. В конце сессии выводится подробная статистика.

## 🚀 Запуск

1. Установите Python (версия 3.8 или новее).
2. Запустите скрипт:
   ```bash
   python main.py
