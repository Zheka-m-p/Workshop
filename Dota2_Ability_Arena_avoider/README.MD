# 🐺 AlphaWolf — Охотник на "задротов" в Ability Arena

Утилита для автоматического поиска игры в Dota 2 (карта Ability Arena) и проверки лобби на наличие нежелательных игроков из топа-100 и твоего личного чёрного списка.  
AlphaWolf анализирует экран в реальном времени, распознает ники через OCR и предупреждает, если в лобби попался «задрот», чтобы ты мог решить, принимать игру или нет.

## 🚀 Быстрый старт (для обычных пользователей)

1.  Скачай `AlphaWolf.exe` из раздела [Releases](https://github.com/Zheka-m-p/Workshop/releases/latest)
2.  Запусти `AlphaWolf.exe` (Windows может показать предупреждение — нажми «Выполнить в любом случае»).
3.  Введи **свой никнейм** в поле «Твой никнейм».
4.  (Опционально) Добавь своих нежелательных игроков через кнопку **➕ Добавить игрока**.
5.  Открой игру Dota 2, запусти Ability Arena и начни поиск матча.
6.  Нажми кнопку **▶️ Старт** в AlphaWolf.
7.  Как только игра найдена, AlphaWolf сделает скриншот лобби, распознает ники и сверит их со списком задротов.
8.  Если в лобби есть нежелательные игроки — появится окно с их списком.
9.  В любой момент сканирование можно остановить кнопкой **⏹️ Стоп**.

## 🛠️ Возможности

-   🔎 Автоматический поиск кнопки «Принять» с оптическим распознаванием текста (Tesseract OCR).
-   👥 Распознавание ников в лобби (русский, английский, китайский языки).
-   📊 Загрузка топ-100 игроков с сайта Ability Arena.
-   ⚙️ Личный чёрный список с сохранением в файл `config.json`.
-   📝 Интерактивный лог: копирование текста (правая кнопка мыши), поиск с подсветкой и навигацией.
-   🔔 Всплывающее окно с результатом проверки (задроты есть / нет).
-   📱 Автоматическая адаптация под текущее разрешение экрана.
-   📦 Полностью портативный EXE — не требует установки Python, Tesseract или библиотек.

## 📋 Системные требования

-   Windows 10/11 (64-разрядная)

## 🔧 Сборка EXE из исходников

Если хочешь собрать программу самостоятельно или внести изменения в код:

1.  Установи зависимости:
    ```bash
    pip install requests pyautogui pytesseract Pillow transliterate pyinstaller
2.  Скопируй папку Tesseract-OCR (с tesseract.exe и папкой tessdata) в корень проекта.

3.  Выполни сборку одной командой:
      ```bash
      pyinstaller --onefile --name "AlphaWolf" --icon=wolf.ico --add-data "Tesseract-OCR;Tesseract-OCR" --noconsole final_try_to_exe_file.py```
4.  Готовый AlphaWolf.exe появится в папке dist.
