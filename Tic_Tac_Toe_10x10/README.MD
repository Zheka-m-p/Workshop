# 👑 Обратные Крестики-Нолики 10x10

Интерактивная игра на Python с использованием библиотеки Pygame. Суть этой версии в том, что вам нужно **избежать** сбора 5 одинаковых символов в ряд — проигрывает тот, кто соберет выигрышную комбинацию по вертикали, горизонтали или диагонали. Последний оставшийся "в живых" — побеждает!

Проект включает два режима: против продвинутого ИИ и классический PvP (игрок против игрока).

## 🎮 Особенности проекта

- **Поле 10х10**: Игра ведется на расширенном поле.
- **Два режима**:
  - `PvE`: Бот с алгоритмом "раздумий" (анализирует ходы и старается избегать ловушек).
  - `PvP`: Локальная игра для двух человек.
- **Удобный Лаунчер**: Главное меню для выбора режимов и выхода.
- **Визуальный фидбек**: Полупрозрачные панели результатов, разделительные линии и адаптивные заголовки окон.
- **EXE-версия**: Проект подготовлен к сборке в один исполняемый файл с собственной иконкой.

## 🛠 Технологии

- **Python 3.9.12**
- **Pygame**: Графический движок.
- **PyInstaller**: Сборка в исполняемый файл.

## 🚀 Как запустить 2 способа:

### I) Скачай готовую программу `Tic-Tac-Toe.exe` из раздела [Releases](https://github.com/Zheka-m-p/Workshop/releases/tag/Tic_Tac_Toe_10x10_v1.01)



### II) Из исходного кода:
1. Установите зависимости:
   ```bash
   pip install pygame
   ```
2. Запустите главный файл:
   ```bash
   python main_menu.py
   ```

### Сборка в .exe:
Для создания автономного файла выполните команду в терминале:
```bash
pyinstaller --clean --noconsole --onefile --icon=tictac.ico --add-data "tictac.ico;." main_menu.py
```
Готовый .exe появится в папке dist.

## 🕹 Управление

- **Мышь**: Выбор клетки на поле.
- **SPACE**: Начать игру заново (после завершения партии).
- **ESC**: Выход из игрового режима в главное меню.

## 📂 Структура файлов

- `main_menu.py`: Точка входа, графический лаунчер.
- `main.py`: Логика игры против компьютера.
- `main_play_pvp.py`: Логика игры для двух игроков.
- `my_algoritms.py`: Библиотека с алгоритмами проверки победы и визуализации матрицы.
- `tictac.ico`: Ресурс иконки приложения.
