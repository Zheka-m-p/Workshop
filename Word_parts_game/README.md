# Word Parts Game

Консольная игра по составлению слов. Вам даётся длинное исходное слово, и ваша задача — составить из его букв как можно больше допустимых «подслов» (не короче трёх букв). Игра загружает словарь с удалённого сервера, поэтому каждый раунд уникален.

## 🎮 Как играть

- Вам предложат составить определённое количество слов из заданного исходного слова.
- Вводите свои варианты. Если слово есть в словаре и не использовалось ранее — оно засчитывается.
- Чтобы закончить игру досрочно, введите `stop`.
- После каждого верного ответа вы можете ввести `значение`, чтобы открыть в браузере определение угаданного слова.

## 🚀 Запуск

1.  Убедитесь, что установлен Python версии 3.9 или новее.
2.  Установите единственную зависимость — библиотеку `requests`:
    ```bash
    pip install requests

3.  Запустите игру:
    ```bash
    python main.py
